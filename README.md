# Movie Watchlist App

## Project Description
Movie Watchlist App is a web application built using HTML, CSS, and Javascript. It allows users to search for movies using the OMDb API and add their favourite movies to a personal watchlist.

## Features Planned
- Search movies by title
- Display movie posters, title, and year
- Add movies to watchlist
- Remove movies from watchlist
- Show empty state when no movies are found
- Show empty state when watchlist is empty

## Future Enhancements
- Sort movies by year
- Filter by type
- Dark mode
- Save watchlist using localStorage
- Loading spinner

## API used
OMDb API

- OMDb API
